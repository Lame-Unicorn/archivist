# Archivist scheduled jobs (Asia/Shanghai)
#
# Install with: crontab scripts/cron/crontab.txt
# Verify with:  crontab -l
#
# Each job runs `archivist digest run*` non-interactively under flock and
# logs to logs/cron/<name>-YYYY-MM-DD.log. A Lark notification is sent on
# success / lock-skipped / failure (configured via lark.notify_user_id in
# config.local.yaml).
#
# IMPORTANT: cron requires absolute paths to the .sh files. Replace
# /path/to/archivist below with the absolute path to your project root
# before running `crontab scripts/cron/crontab.txt`.
#
# The wrapper scripts auto-detect their own location, so no extra env vars
# are needed at runtime.

CRON_TZ=Asia/Shanghai

# Daily digest — Tue-Sat at 09:00, aligned with ArXiv publish cycle.
# ArXiv publishes at ET 20:00 (= Beijing 08:00 next day), so Tue 09:00
# catches Monday night's batch (which includes weekend submissions).
# Tuesday's digest covers Fri+Sat+Sun+Mon via _coverage_range().
0 9 * * 2-6  /path/to/archivist/scripts/cron/daily-digest.sh

# Weekly digest — every Tuesday at 09:30 (covers the previous ISO week,
# runs after daily digest to include all 5 daily reports)
30 9 * * 2   /path/to/archivist/scripts/cron/weekly-digest.sh

# Monthly digest — 1st of every month at 10:00 (covers the previous month)
0 10 1 * *   /path/to/archivist/scripts/cron/monthly-digest.sh
